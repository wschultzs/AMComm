# Instruções para rodar

- Realizar o clone do projeto
- Rodar o comando npm install para instalar as dependências
- Rodar o comando ng s para iniciar a aplicação

## Tecnologias utilizadas

- Angular v.15
- NG-Bootstrap v.14.0.1
